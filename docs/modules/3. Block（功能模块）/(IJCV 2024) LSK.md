# (IJCV 2024) LSK

## 1. 模块简介
- **论文地址**: [https://github.com/zcablii/Large-Selective-Kernel-Network](https://github.com/zcablii/Large-Selective-Kernel-Network)
- **源文件**: `(IJCV 2024) LSK.py`

### 设计机制
- 深度可分离卷积，保持输入和输出通道数一致，卷积核大小为5
- 空间卷积，卷积核大小为7，膨胀率为3，增加感受野
- 1x1卷积，用于降维
- 结合平均和最大注意力的卷积
- 最后的1x1卷积，将通道数恢复到原始维度
- 对输入进行两种不同的卷积操作以生成注意力特征
- 对卷积特征进行1x1卷积以降维
- 将两个特征在通道维度上拼接
- 计算平均注意力特征
- 计算最大注意力特征
- 拼接平均和最大注意力特征
- 通过卷积生成注意力权重，并应用sigmoid激活函数
- 根据注意力权重调整特征
- 最终卷积恢复到原始通道数
- 通过注意力特征加权原输入

block = LSKblock(64).cuda()  # 实例化LSK模块并转移到GPU
